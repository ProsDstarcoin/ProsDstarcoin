# ProsDstar Coin Website Package

Files included:
- index.html
- style.css
- scripts.js
- assets/logo.png
- whitepaper/PSD_Whitepaper_v1.pdf

Instructions:
1. Create a new GitHub repository.
2. Upload all files preserving the folder structure.
3. In GitHub, go to Settings -> Pages and enable publishing from the main branch.
4. Your site will be available at https://yourusername.github.io/reponame/

Replace the placeholder logo (assets/logo.png) with your real logo file if desired.
